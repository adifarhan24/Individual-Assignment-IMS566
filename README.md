# EduCentre 

This is a polished, responsive, and accessible single-repo static web application that demonstrates:
- Authentication (simulated)
- Responsive navigation
- Dashboard with charts (Chart.js)
- Data views: Students & Courses (search, add, export CSV)
- Accessible HTML and semantic layout
- Clean code separation (HTML / CSS / JS modules)

## How to run locally
1. Clone or copy files to a folder.
2. Open `index.html` in Chrome (recommended).
3. Demo login credentials:
   - Email: `admin@gmail.com`
   - Password: `password123`
4. Add/edit students from Students page (data persists in `localStorage`).
